Install your node package from this command

install npm

build your app using this command

ng build

check your angular version

ng -v

make the new component in angular app than this command you will be fire

ng g c component-name

make the new Service in angular app than this command you will be fire

ng g s service-name

if you want to update your context then fire this command

fire this command from ERP.Data, choose when fire command in package manager console

Scaffold-DbContext "Data Source=.;Initial Catalog=ERP;Trusted_Connection=True;MultipleActiveResultSets=true" Microsoft.EntityFrameworkCore.SqlServer -OutputDir Models -f
